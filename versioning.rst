Versioning
##########

Release Criteria
****************

    .. code-block:: text

        v1.2.3a
         | | ||___ Support for a new FOS version is added to FOS-ansible
         | | |____ Bug fix is released for FOS-ansible
         | |______ When new features are added to FOS-ansible
         |________ Changes when a release is backward in-compatiable with the previous version
